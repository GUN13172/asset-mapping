## [1.0.0] - 2026-02-06

### 新增
- ✨ 支持 Hunter、FOFA、Quake、DayDayMap 四大平台
- ✨ API 密钥管理功能
  - 多密钥支持
  - 密钥验证
  - 密钥轮换
  - 配额查询
- ✨ 资产查询功能
  - 多平台查询
  - 分页查询
  - 时间范围筛选
  - 实时进度显示
- ✨ 查询语法转换
  - 平台间语法互转
  - 智能字段映射
  - 运算符转换
- ✨ 数据导出功能
  - CSV 格式导出
  - 批量导出
  - 断点续传
  - 进度显示
- ✨ 历史记录管理
  - 查询历史保存
  - 历史记录搜索
  - 历史记录导出
  - 一键重新查询
- ✨ 智能查询联想
  - 语法提示
  - 字段联想
  - 运算符提示
- ✨ 主题切换
  - 浅色主题
  - 深色主题
  - 跟随系统
- ✨ 多语言支持
  - 简体中文
  - 英文
