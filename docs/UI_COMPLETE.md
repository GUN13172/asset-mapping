## 🚀 使用说明

### 切换主题
1. 打开"设置"页面
2. 找到"界面设置"部分
3. 在"主题"下拉框中选择：
   - 浅色（默认，明亮清新）
   - 深色（暗色护眼）
   - 跟随系统（自动切换）
4. 点击"保存设置"

### 自定义主题
编辑以下文件来自定义配色：
- `src/styles/theme-light.css` - 亮色主题
- `src/styles/theme.css` - 暗色主题

### 组件样式
所有组件自动继承主题样式，无需额外配置。
