## 使用方式

### AssetQuery 组件
1. 输入查询条件
2. 点击"查询"按钮 → 显示搜索进度
3. 查询完成后点击"导出结果" → 显示导出进度

### ExportData 组件
1. 选择导出类型
2. 输入查询条件
3. 设置导出参数
4. 点击"开始导出" → 显示导出进度

## 技术要点

- 使用 Tauri 事件系统 (`listen`)
- 实时接收后端进度事件
- 自动更新 UI 状态
- 组件卸载时清理监听器
